# 🍽️ Pantrix – Smart Pantry Recipe Recommender

Pantrix is a desktop application built with Python Tkinter and MySQL that helps users make the most of the ingredients already available in their pantry. Instead of wondering what to cook, users can simply add their available ingredients and receive personalized recipe recommendations instantly.

## ✨ Features

### 🔐 User Authentication
- Secure Login and Signup system
- Personalized user accounts

### 🥫 Pantry Management
- Add and manage ingredients available in your pantry
- Keep track of what ingredients you have on hand

### 👨‍🍳 Smart Recipe Recommendations
- Get recipe suggestions based on your available ingredients
- Discover meals you can prepare without unnecessary grocery trips

### 📋 Missing Ingredient Detection
- View ingredients required for a recipe that are not currently available in your pantry
- Easily identify what needs to be purchased

### 🛒 Shopping List Generator
- Add missing ingredients directly to your shopping list
- Organize grocery shopping efficiently

### ❤️ Favorite Recipes
- Save and manage your favorite recipes
- Quickly access recipes you love to make again
## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- MySQL (Database Management)

## 🎯 Project Goal
Pantrix aims to reduce food waste, simplify meal planning, and make cooking more convenient by intelligently connecting pantry ingredients with suitable recipes.
